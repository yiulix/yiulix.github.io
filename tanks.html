Tanks

This is a 3-player turn-based tank fight game. This is the game I dreamed when I was a a child.

  • Demo
  • Gameplay
  • Credits

Goal:

  • Defeat enemy tanks and be the last one standing.

Mechanics:

  • Three different tanks
  • Using "F1", "F2", "F3" to select cannons. More powerful cannons require more flue.
  • Using "Q", "W", "E", "R" to move, using up and down arrow keys to aim, and hold space bar to shoot.

Technical Feature:

  • Dynamic terrain: By calculating the terrain heightmap in realtime, the cannons can leave explosion holes on the map.
